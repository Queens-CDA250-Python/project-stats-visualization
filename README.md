# Project Stats Visualization

In this project check-in you will perform variable analysis and represent the analysis with accompanying visualizations.

Criteria:

- Perform one Pearson r correlation test on your data.
- Perform on t-test on your data.
  - Include any accompanying tests necessary for interpretation.
- Perform one ANOVA test on your data.
  - Include any accompanying tests necessary for interpretation.
- Explain why you chose the variables and interpret your results.
- Include at least one visualization for each test that you conduct to demonstrate the results.
  - Include explanations for why you chose the visualizations you did.
- Include a narrative that interprets the results.
- The narrative should interweave the explanation for the variable choices, the visualization, and the interpretation.
